# CSVアンケート分析ツール

ブラウザだけで動く、静的な CSV アンケート分析ツールです。  
CSV を読み込むと、列ごとの簡易分析、グラフ表示、PNG 保存、PDF 出力ができます。

## 主な機能

- CSVファイルの読み込み
- CSVテキスト貼り付け読み込み
- 1行目ヘッダー認識
- 列タイプの自動判定
  - カテゴリ
  - 数値
  - 自由記述
  - 非表示
- 列タイプの手動変更
- グラフ形式切り替え
- 各カードの PNG 保存
- 全体 PDF 出力

## 想定用途

- 演劇公演アンケート
- イベントアンケート
- ワークショップ集計
- セミナーや講演会の簡易集計

## 使い方

1. `index.html` をブラウザで開きます
2. CSVファイルを選ぶか、CSVテキストを貼り付けます
3. `読み込む` を押します
4. 列ごとにタイプやグラフ形式を調整します
5. 必要に応じて PNG 保存または PDF 出力を行います

## ファイル構成

- `index.html`
  - 画面レイアウトと CDN 読み込み
- `style.css`
  - カード UI とレスポンシブ表示
- `script.js`
  - CSV 読み込み、列判定、集計、グラフ描画、PNG/PDF 出力
- `sample.csv`
  - 動作確認用サンプル

## 使用ライブラリ

- [Papa Parse](https://www.papaparse.com/)
- [Chart.js](https://www.chartjs.org/)
- [html2canvas](https://html2canvas.hertzen.com/)
- [jsPDF](https://github.com/parallax/jsPDF)

すべて CDN 読み込みです。ビルドツールは使っていません。

## 対応している CSV

- UTF-8 想定
- 1行目ヘッダー
- カンマ区切り中心
- タブ区切り、セミコロン区切りも簡易対応

## 列判定の考え方

簡易判定です。誤判定は起こり得るため、手動変更 UI を用意しています。

- 数値列
  - 空欄を除いた値の大半が数値
- 自由記述列
  - ユニーク値が多い
  - 平均文字数が長い
- カテゴリ列
  - 上記以外

## 既知の制限

- 日本語の頻出語抽出は簡易方式です
- 頻出語は参考表示であり、厳密な自然言語解析ではありません
- 高度な自然言語解析はしていません
- 特殊な CSV 形式や複雑な引用符付きデータには弱い場合があります
- PDF は帳票厳密性より読みやすさ優先です
- 大量データでは PNG / PDF 出力に時間がかかる場合があります

## ローカル完結

- サーバー通信はしません
- データベースは使いません
- ログイン機能はありません
